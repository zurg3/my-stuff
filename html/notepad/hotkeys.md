# Keyboard Shortcuts

| Action                              | Shortcut               |
| ----------------------------------- | ---------------------- |
| Tabulation                          | `Tab`                  |
| Insert current time/date            | `F5`                   |
| Open file                           | `Ctrl` + `O`           |
| Open file from URL                  | `Ctrl` + `Shift` + `O` |
| Save file                           | `Ctrl` + `S`           |
| Remove keyboard focus from textarea | `Ctrl` + `Q`           |
| Execute                             | `Ctrl` + `E`           |
| Toggle write/read mode              | `Ctrl` + `H`           |
| Toggle word wrap                    | `Ctrl` + `J`           |
| Reset Notepad settings              | `Ctrl` + `K`           |
| Minify text                         | `Ctrl` + `M`           |
| Decrease font size                  | `Ctrl` + `[`           |
| Increase font size                  | `Ctrl` + `]`           |
| Switch to previous theme            | `Ctrl` + `<`           |
| Switch to next theme                | `Ctrl` + `>`           |
